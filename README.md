# i3status_stock_ticker
A stock quote ticker for use with <a href="https://github.com/ultrabug/py3status">py3status.</a> 

<pre>
ticker_google "s&p"{
    exchange = "INDEXSP"
    symbol = ".INX"
    cache_timeout = 2
}

ticker_google "dji"{
    exchange = "INDEXDJX"
    symbol = ".DJI"
    cache_timeout = 2
}

</pre>
